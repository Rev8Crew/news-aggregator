# News

## Api
POST /api/add - добавляет новостьж<br>
* title - заголовок
* text - текст новости
* url - url новости
* release_at timestamp
* category - название категории.
* rss - url rss сайта.
***
GET /api/list - возвращает список всех новостей в json виде.<br>
* Параметры:
* filter - category|date|rss|today|month|week
* order - asc|desc
